## Useful commands

### setup frontend
```sh
cd web
npm install
npm run watch # it will be watching for changes and rebuilding the app to /app/build
```

### setup backend
```sh
virtualenv venv
source venv/bin/activate
```

```sh
cd app
pip install -r requirements.txt
```

```sh
flask run # it will serve the /app/build
```
